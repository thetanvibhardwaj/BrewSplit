# BrewSplit

BrewSplit is a shared expense management application inspired by Splitwise.

The application helps users:

* Manage shared expenses
* Track balances
* Handle settlements
* Import expense data from CSV
* Detect anomalies in imported data
* Support changing group memberships
* Support multiple currencies

## Tech Stack

Frontend:

* React
* TypeScript
* React Router

Backend:

* Node.js
* Express.js

Database:

* PostgreSQL

Deployment:

* Vercel (Frontend)
* Render (Backend)
* Neon PostgreSQL (Database)
